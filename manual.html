Step 02
添加到桌面（可选但推荐）

将 App 添加到桌面后，可像原生应用一样使用，没有浏览器工具栏干扰。

🍎 Mac · Safari
1. 用 Safari 打开链接
2. 顶部菜单 → 文件
3. 点击 添加到程序坞
4. 从 Dock 直接打开
🪟 Windows · Chrome
1. 用 Chrome 打开链接
2. 地址栏右侧点 安装图标
3. 或菜单 → 安装此网站为应用
4. 桌面出现快捷方式
📱 iPhone · Safari
1. 用 Safari 打开链接
2. 底部点击 分享按钮 □↑
3. 选择 添加到主屏幕
4. 像 App 一样出现在桌面
🤖 Android · Chrome
1. 用 Chrome 打开链接
2. 右上角点 ⋮ 菜单
3. 选择 添加到主屏幕
4. 桌面出现快捷方式

功能介绍 · Features

✅
Feature 01
任务清单
+

点击右上角 ＋ 按钮 新建任务，填写任务名称，可选填截止时间和优先级

✓

点击任务右侧的 ✓ 标记完成，任务自动置灰并划线

✕

点击任务右侧的 ✕ 删除任务

🔴 紧急 · 🟡 重要 · 🟢 普通，三级优先级一目了然
截止时间越近，自动显示倒计时提醒（如"24分钟后 快跑！"）
任务栏左边缘可拖拽调整宽度，满足不同使用习惯
💾 自动保存：所有任务保存在你的浏览器本地，关闭后重新打开依然存在。
🧠
Feature 02
专注计时器
打开 App 后自动开始 60 分钟倒计时
倒计时结束时，小恐龙会醒来并弹出提醒：该站起来拉伸了！
拉伸完毕后点击 "拉伸完成 / 重置" 按钮，计时重新开始
今日专注次数自动累计，每天零点清零
🌤
Feature 03
天气 · 时区时钟
1

点击天气卡片右上角的 ✏️ 图标

2

输入城市名（支持中文如"北京"，或英文如"Shanghai"）

3

从搜索结果列表中选择你的城市，顶部时钟自动切换为该城市时区

📝
Feature 04
随想录
位于页面左下角，随时记录灵感、想法、备忘
最多支持 500 字，右上角实时显示剩余字数
边写边自动保存，关闭页面后内容不丢失
🎨
Feature 05
壁纸换色
点击 Wallpaper 按钮，调色盘选择你喜欢的颜色
背景自动生成与所选颜色匹配的渐变壁纸
点击 重置 按钮恢复默认深蓝背景
💾
重要说明
关于你的数据
所有数据（任务、随想录、城市、壁纸）保存在 你自己的浏览器 里
开发者和其他用户 完全看不到 你的任何内容
每位用户的数据相互独立，互不影响
⚠️ 注意：数据存在当前浏览器本地，换设备或清除浏览器缓存后数据会丢失。建议固定在同一台设备、同一浏览器上使用。
