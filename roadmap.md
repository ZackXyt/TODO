## v2.0 · 三大核心功能（进行中）

### ① 「我的一天」☀️（优先级：最高）
> 每天主动规划今日任务，形成聚焦清单

- [ ] 任务数据增加 `myDay` 字段
- [ ] 任务列表顶部增加「全部 / 我的一天」视图切换
- [ ] 每张任务卡片增加☀️图标，一键加入/移出「我的一天」
- [ ] 「我的一天」视图显示晨间问候语 + 今日任务数
- [ ] 每日 0 点自动清空「我的一天」（可选）

### ② 子任务（步骤）📋（优先级：高）
> 复杂任务可拆解为多个小步骤，逐一完成

- [ ] 任务数据增加 `steps: []` 字段
- [ ] 任务卡片可展开，显示子步骤列表
- [ ] 支持新增 / 勾选 / 删除子步骤
- [ ] 子任务完成率显示在主任务卡片上（如 2/5）

### ③ 多列表 / 分类 📁（优先级：中）
> 工作、学习、生活……任务按场景归类

- [ ] 支持创建 / 重命名 / 删除自定义列表
- [ ] 任务归属某个列表（默认「收集箱」）
- [ ] 列表切换侧边栏或顶部 Tab
- [ ] 不同列表可设置独立颜色标记
